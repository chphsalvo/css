<h2>Run project</h2>
<pre>
  <code>
    # Install
    git clone https://github.com/chphsalvo/front-end-framework.git
    cd front-end-framework
    npm install
    <br>
    # Watch
    npm run watch
   	<br>
    # Deploy
    npm run build
  </code>
</pre>
